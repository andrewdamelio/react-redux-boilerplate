## Installation

You'll need to have [Node.js] (https://nodejs.org/) to get started.

```bash
$ npm install # Install dependencies
```

* If your having issues try updating to the latest version of node.


## Getting Started

#### Tests
```bash
$ npm run test
```

#### Dev
```bash
$ npm start
```
Open http://localhost:8080 in your browser.

#### Build
```bash
$ npm run clean
$ npm run build
$ http-server -p 8080 .

```
Open http://localhost:8080 in your browser.
